# Service Mesh Observer

Service mesh observability for Istio and Linkerd: traffic flow, latency, error monitoring.

## Features
- Traffic flow visualization
- Latency percentile tracking (p50/p95/p99)
- Error rate monitoring
- mTLS certificate status

## Integrations
- Istio (Envoy sidecar metrics)
- Linkerd (Prometheus scrape)
- Prometheus + Grafana dashboards
